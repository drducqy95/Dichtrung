# Phase 02: Khởi tạo Vbook Extension Project

Status: ⬜ Pending
Dependencies: phase-01-dichtrung-automation.md

## Objective
Tạo không gian làm việc VS Code cho Extension mới dựa trên tool `vbook-extension-maker-master`, thiết lập các thành phần hệ thống cơ bản.

## Requirements

### Functional
- [ ] Tạo folder dự án `d:\APP\ext-dichtrung`.
- [ ] Thiết lập `plugin.json` định dạng chuẩn.
- [ ] Móc nối dự án vào `vbook-extension-maker-master` để có thể sử dụng Live Testing & Vitest.

## Implementation Steps
1. [ ] Tạo folder `ext-dichtrung`.
2. [ ] Copy icon (icon của Dichtrung) vào folder extension.
3. [ ] Tạo file `plugin.json` với metadata phù hợp.
4. [ ] Tạo trước file khung `home.js`, `detail.js`, `toc.js`, `chap.js`.

## Files to Create/Modify
- `d:\APP\ext-dichtrung\plugin.json`
- Các file `.js` trống.

## Test Criteria
- [ ] Dự án báo lỗi thiết lập thành công.
